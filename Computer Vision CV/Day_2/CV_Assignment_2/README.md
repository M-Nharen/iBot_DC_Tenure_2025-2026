# Circle Detector

This is a Python script used to detect circles in any image.

## Overview:

This directory contains all the necessary steps to setup and run the pencil_sketch.py program

```
Directory  Structure

├── test

|    ├── test2.jpg

├── circle_detector.py
└── README.md
```

## Setup:

The program requires the following libraries:

```
1.opencv-python
2.numpy
3.matplotlib
4.sys
5.Pillow
```
## Usage:

To start the program run

```
python3 circle_detector.py <input_file_path> <output_file_path>
```

```
Run the program along with the image file location and output file location in the terminal.

Ex: python3 circle_detector.py home/test1.jpg home/result1.jpg
```
